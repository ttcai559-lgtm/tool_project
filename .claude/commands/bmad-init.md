# BMad 项目初始化

## 工作流说明

1. 读取工作流配置：`{project-root}/.bmad/bmm/workflows/workflow-status/init/workflow.yaml`
2. 读取工作流指令：`{project-root}/.bmad/bmm/workflows/workflow-status/init/instructions.md`
3. 读取核心工作流引擎：`{project-root}/.bmad/core/tasks/workflow.xml`
4. 按照指令执行工作流，初始化项目

请执行 workflow-init 工作流以初始化新的 BMad Method 项目。
