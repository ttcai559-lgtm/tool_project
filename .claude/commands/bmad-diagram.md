# 创建系统架构图

使用 Excalidraw 格式创建系统架构图、ERD、UML 图或通用技术图表。

## 工作流说明

1. 读取工作流配置：`{project-root}/.bmad/bmm/workflows/diagrams/create-diagram/workflow.yaml`
2. 读取工作流指令：`{project-root}/.bmad/bmm/workflows/diagrams/create-diagram/instructions.md`
3. 读取检查清单：`{project-root}/.bmad/bmm/workflows/diagrams/create-diagram/checklist.md`
4. 读取 Excalidraw 库：`{project-root}/.bmad/bmm/workflows/diagrams/_shared/excalidraw-library.json`
5. 创建 Excalidraw 格式的技术图表

请执行 create-excalidraw-diagram 工作流，创建技术图表。
