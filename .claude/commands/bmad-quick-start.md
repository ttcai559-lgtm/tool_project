# BMad 快速开始

BMad Method 快速入门指南。

## 步骤 1: 初始化项目

如果这是新项目，运行：
```
/bmad-init
```

## 步骤 2: 检查状态

随时检查"我现在应该做什么？"：
```
/bmad-status
```

## 步骤 3: 按阶段进行

### 分析阶段 (Phase 1)
- `/bmad-brainstorm` - 项目头脑风暴
- `/bmad-research` - 市场/技术研究
- `/bmad-product-brief` - 产品简介

### 规划阶段 (Phase 2)
- `/bmad-prd` - 产品需求文档（适用于大型项目）
- `/bmad-tech-spec` - 技术规范（适用于快速流程）
- `/bmad-ux-design` - UX 设计

### 解决方案阶段 (Phase 3)
- `/bmad-arch` - 架构设计
- 创建史诗和故事

### 实施阶段 (Phase 4)
- `/bmad-sprint-plan` - Sprint 规划
- `/bmad-dev-story` - 开发故事
- `/bmad-code-review` - 代码审查

## 快速参考
