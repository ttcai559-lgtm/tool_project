# 创建数据流图

使用 Excalidraw 格式创建数据流图 (DFD)。

## 工作流说明

1. 读取工作流配置：`{project-root}/.bmad/bmm/workflows/diagrams/create-dataflow/workflow.yaml`
2. 读取工作流指令：`{project-root}/.bmad/bmm/workflows/diagrams/create-dataflow/instructions.md`
3. 读取检查清单：`{project-root}/.bmad/bmm/workflows/diagrams/create-dataflow/checklist.md`
4. 创建 Excalidraw 格式的数据流图

请执行 create-excalidraw-dataflow 工作流，可视化数据流和系统交互。
